🤖 CONTROL DERIVATIVO — INTERFAZ ALTERNA

Kd = Ganancia Derivativa · Anticipación

🔁 ECUACIÓN DEL CONTROL DERIVATIVO

La señal de control es proporcional a la velocidad de cambio del error.

u(t) = Kd · de(t)/dt

Donde de/dt indica qué tan rápido varía el error.

⚡ EFECTO: ANTICIPACIÓN

Actúa antes de que el error crezca: si el error cambia rápido, aplica una corrección fuerte para frenar o acelerar.

Funciona como un amortiguador que reduce overshoot y oscilaciones.

⚠️ PRECAUCIONES

• Amplifica el ruido (por eso se filtra en la práctica).
• No se usa solo; suele combinarse con P o PI.
• Ideal para sistemas con inercia.

🎮 APLICACIÓN AQUÍ

El robot debe posicionarse rápido. La acción derivativa detecta la velocidad de aproximación y aplica freno antes del setpoint.
